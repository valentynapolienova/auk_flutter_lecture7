# lecture5

Multi-page pet adoption app with home, detail, and settings page.

### Steps
1. Take a look at our future app structure:
   ```
   /lib
   main.dart
     /screens
       home_screen.dart
       pet_detail_screen.dart
       settings_screen.dart
   /widgets
       pet_card.dart
    /models
       pet.dart
   ```
2. Configure app in `main.dart` (theme and routes).
3. Create `Pet` model in `models/pet.dart`.
4. Create `PetCard` widget in `widgets/pet_card.dart`.
5. Create `HomeScreen` in `screens/home_screen.dart`.
6. Create `PetDetailScreen` in `screens/pet_detail_screen.dart`.
7. Create `SettingsScreen` in `screens/settings_screen.dart`.
8. Optional: Add google fonts package for custom fonts. (https://pub.dev/packages/google_fonts)
